6 提纲

6.1 提纲号

以数字和点组成,通过空格与提纲名分隔,例如:

6.1.1 提纲号示例

点不能出现在开头或结尾,例如

.6.1.2 错误示范

6.1.3. 错误示范

不能出现两个及以上连续的点,例如:

6..1…4 错误示范

提纲号会被自动配置为锚点,可直接使用{6}{6.1}
